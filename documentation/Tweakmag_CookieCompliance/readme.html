Overview

The Tweakmag Dynamic Content extension enhances the default Magento product functionality by allowing you to tags in the form of {tag} which gets replaced automatically with the products data. For example, by adding {name} in the meta description field will get replaced with the actual product name automatically.

You can add tags to the following:

  • Product Meta Title *
  • Product Meta Keywords *
  • Product Meta Description *
  • Product Description

For the items marked with an * above you can set a template that will be used for all products (but still override at the product level). Please read the Configuration Section below to learn how to do this.

Configuration

It is advised that you first set up the default templates that will apply for all products that do not have specific content provided. The following templates are supported:

  • Default Title: System > Confguration > General > Design > HTML Head
  • Default Meta Description: System > Confguration > General > Design > HTML Head
  • Default Meta Keywords: System > Confguration > General > Design > HTML Head

To set up the Default Meta Description for example you could enter something like:

The {color} {name} from {manufacturer} is only {price}

And it will automatically pull in the product information, for the example the above might look like:

The red Leather Handbag from Guicci is only $450.00

These templates can be configured for each website and/or store views you have setup as well. You will usually only need to do this if you have different languages.

For an individual product you can apply the same logic as above to override the default templates with even more specific dynamic content.

The tags you can use is only limited to what attributes are available to the product including all the attributes you create yourself. The tags are the same as the attribute code which you can view by navigating to:

Catalog > Attributes > Manage Attributes
